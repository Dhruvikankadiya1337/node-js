## Project Description
This project is a backend REST API built using **Node.js, Express, MongoDB, and Mongoose**.
The API allows students to perform various **GET operations** on a product collection such as
fetching products, searching, filtering, sorting, and pagination.

No frontend is used in this project. All APIs are tested using **Thunder Client / Postman**.

---

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- ES Module (import / export)
- Thunder Client (for API testing)
